# Document Management Dashboard

A full-stack web application for uploading and managing company PDF documents with real-time progress tracking and notifications.

## Features

- Upload single or multiple PDF files
- Real-time upload progress bars
- Backend file storage using Multer
- Notification system for uploads
- Uploaded documents table
- Modern responsive dashboard UI
- Bulk upload support

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- Multer

## Project Structure

/document-dashboard
/client → React frontend
/server → Express backend
/server/uploads → Uploaded PDF files

## Run Frontend

```bash
cd client
npm install
npm run dev
```

## Run Backend

```bash
cd server
npm install
node server.js
```
